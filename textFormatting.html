Text Formatting

Text formatting tags are used to define the style and appearance of text in an HTML document.

The strong tag is used to define important text. It is typically displayed in bold.

The bold tag is used to define bold text. It is typically displayed in bold.

The emphasis tag is used to define emphasized text. It is typically displayed in italics.

The italic tag is used to define italic text.

The underline tag is used to define underlined text.

The mark tag is used to define marked or highlighted text. It is typically displayed with a yellow background.

The small tag is used to define smaller text. It is typically displayed in a smaller font size than the surrounding text.
